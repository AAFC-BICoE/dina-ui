# dina-ui

React based user interface for AAFC-DINA.

## Required

- NodeJS
- Yarn

## Install the Package

Install dependencies for the parent package and sub-packages using the following yarn command at the top-level directory:

`yarn`

## Generate docs

From the top-level directory run:

`yarn docs`
