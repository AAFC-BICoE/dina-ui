> [!IMPORTANT]
>
> ### 🚀 Getting Started
>
> **Run and develop DINA locally using [dina-local-deployment](https://aafc-bicoe.github.io/dina-local-deployment/#_developer_environment_setup).**

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** - Version 20
- **Yarn** - Version 1.22

## 📚 Documentation

- **[Local Deployment Documentation](https://aafc-bicoe.github.io/dina-local-deployment/#_developer_environment_setup)** - Local Developer Environment Setup
- **[DINA User Guide](https://aafc-bicoe.github.io/dina-documentation/)** - End-user documentation
- **[dina-ui Technical Docs](https://aafc-bicoe.github.io/dina-ui/)** - Developer documentation

## 🔧 Installation

Install all dependencies for the parent package and sub-packages:

```bash
yarn
```

Run this command from the top-level directory.
